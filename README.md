# Neo Tasks Repository

این ریپازیتوری شامل **تمام پروژه‌ها و آزمون‌هایی است که توسط هلیا صداقتیان برای مجموعه نئو طراحی و توسعه داده شده‌اند**.  

## توضیح کلی
- هر پروژه یک آزمون یا برنامه روانشناسی/شخصیتی است.
- فایل‌ها شامل کدهای برنامه (`.py`) و فایل‌های مرتبط با خروجی یا داده‌های مورد استفاده هستند.
- هدف از این ریپازیتوری، **ثبت و نگهداری پروژه‌های شخصی مرتبط با مجموعه نئو** است.

## فهرست پروژه‌ها

1. **Academic Motivation & Wellbeing Test**
   - بررسی انگیزه و رفاه تحصیلی دانش‌آموزان
   - فایل‌های اصلی: 
     - `Academic Motivation & Wellbeing Test.py`
     - `results_academic_psych.csv`

2. **Educational Psychology Test**
   - آزمون روانشناسی آموزشی
   - فایل‌های اصلی: 
     - `educational_psych_test.py/educational_psych_test.py`
     - `results.csv`

3. **MBTI Test**
   - آزمون شخصیت MBTI
   - فایل اصلی: `MBTI-test.py`

4. **Optimism and Pessimism Personality Test**
   - ارزیابی خوش‌بینی و بدبینی
   - فایل اصلی: `Optimism and Pessimism Personality Test.py`

5. **Test Anxiety Inventory (TAI)**
   - آزمون اضطراب امتحان
   - فایل اصلی: `Test Anxiety Inventory(TAI).py`

## نحوه استفاده
- همه برنامه‌ها با پایتون نوشته شده‌اند و قابل اجرا در محیط Python هستند.
- فایل‌های CSV به عنوان نمونه داده یا خروجی برنامه در پروژه‌ها موجود است.
- برای اجرای هر آزمون کافی است فایل `.py` مربوطه را اجرا کنید.

## نکات
- این ریپازیتوری صرفاً برای نگهداری پروژه‌ها و آموزش‌های انجام شده برای مجموعه نئو است.
- فایل‌های خروجی (`.csv`) را می‌توان بعد از اجرای برنامه دوباره تولید کرد.
